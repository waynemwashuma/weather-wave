# Weather Wave

## Quick Start

This repository contains:

- A React frontend in `frontend/`
- An Express + PostgreSQL backend in `backend/`

1. Install dependencies from the repository root:

```bash
npm install
```

2. See the [backend guide](./backend/README.md) to set up the backend.
3. Start both apps:

```bash
npm start
```

Frontend:

- `http://localhost:5173`

Backend:

- `http://localhost:8000`
